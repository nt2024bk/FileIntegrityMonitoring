# File Integrity Monitoring Scripts

This repository contains shell scripts for basic file integrity monitoring on Linux systems, including automated checks, baseline updates, and email reporting.

## Scripts
1. **integrity_check.sh**: Monitors file integrity and generates a report.
2. **update_baseline.sh**: Updates the integrity baseline file.
3. **email_report.sh**: Sends the integrity report via email.
4. **fim_cron_setup.sh**: Sets up cron jobs for automation.

## Requirements
- Linux-based system
- `sha256sum` utility
- `mail` utility (for email reporting)
